## ImaDOG
타겟
- 반려견을 가족같이 생각하는 경제력 있는 싱글

기능
- 웨어러블 개목줄과 연동 -> 산책시 잃어버렸을 때 대비 위치추적
- 웨어러블 개목줄과 연동 -> 강아지 건강상태 체크
- 예방접종 시기 알림
- 주변 동물병원, 애견까페 추천
- 개 옷, 장난감, 간식 추천, 리뷰
- 개 종류별로 사진 올릴 수 있고 자유게시판
- Home CCTV를 통해 외출 시 강아지 확인 가능

핵심 사항
- GPS 기능과 건강상태를 체크할 수 있는 강아지용 웨어러블 기기 기술이 핵심


* 인스타 + 자신의 개 사진(추억쌓기) -> 공개/ 비공개 설정 가능 + 미아방지 + 건강관리 + 홈 cctv

* 인스타(SNS)인데 개 입장에서 쓰는거지.
ex) 아 오늘 주인이 산책 안시켜줌 ㅡㅡ

* 웨어러블 건강검진은 아직 기술적 문제..

* 산책시 미아방지는 리디어블 팔찌 참고하여 이쁜 디자인 목줄 같은거 해서 판매

* 동물병원 -> 즐겨찾기 등록 가능 -> 병원과 Q&A 가능 (?)

* 애완동물 종류별로 확장

* 처음에 고양이 같은 좀 그런걸로 시작해서 확장..
